## setw() 函数

`setw()` 函数可以控制输出流的**下一个**输出内容的场宽。

以下是一个使用的例子。

当运行以下语句时：

```cpp
int a=114514;
cout<<a;
```



输出结果将为：

```plain
114514
```



当包含了头文件`<iomanip>`，并运行以下语句时：

```cpp
int a=114514;
cout<<setw(10)<<114514;  //设置场宽为10
```



输出结果将为：

```plain
    114514
```



可见，当**下一个**输出的内容的宽度不足 `setw()` 函数中设置的参数时，输出流将自动在这个输出内容的**前面**添加字符（默认为空格）来补齐宽度。

当下一个输出的内容超过 `setw()` 函数中设置的参数时，将不会产生任何效果，例如：

```cpp
int a=114514;
cout<<setw(1)<<a;
```



输出结果将为：

```plain
114514
```



**需要特别注意， `setw()` 函数只会对下一个输出内容有效，对于多个输出内容，需要重新设置，例如：**

```cpp
int a=114,b=514;
cout<<setw(10)<<a<<" "<<b;
```



输出结果将为：

```plain
       114 514
```



正确的使用方法为：

```cpp
int a=114,b=514;
cout<<setw(10)<<a<<" "setw(10)<<b;
```



输出结果将为：

```plain
       114        514
```



**2.B2006 地球人口承载力估计******(每个单位时间新生 *q* 个单位的资源。)****

假设地球上的新生资源按恒定速度增长。照此测算，地球上现有资源加上新生资源可供 *x* 亿人生活 *a* 年，或供 *y* 亿人生活 *b* 年。

为了能够实现可持续发展，避免资源枯竭，地球最多能够养活多少亿人？

## 解法：

假设现有资源是k;消耗ax;或者消耗by,新生aq;

即k+aq-ax=k+bq-by

(a-b)q=ax-by

q=(ax-by)/a-b
